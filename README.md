## Ошибка №1

**Что пишет компьютер (по-английски):**
`Extra bracket`

**Что пишет компьютер (по-русски):**
Лишняя скобка

**Что это значит простыми словами:**
Вы поставили скобку там, где она не нужна. Возможно, вы перепутали круглые скобки `()` с квадратными `[]` или фигурными `{}`.

**Как исправить:**
1. Проверьте все открывающие и закрывающие скобки в строке.
2. Убедитесь, что количество открывающих скобок равно количеству закрывающих.
3. Проверьте, правильно ли вы используете типы скобок:
   - `( )` — для вызова методов и группировки выражений
   - `[ ]` — для обращения к элементам массива

**Неправильно:**
```basic
x = (5 + 3))   ' лишняя закрывающая скобка
```

**Правильно:**
```basic
x = (5 + 3)    ' одна открывающая и одна закрывающая
```

## Ошибка №2

**Что пишет компьютер (по-английски):**
`Brackets not closed properly`

**Что пишет компьютер (по-русски):**

**Что это значит простыми словами:**
Вы открыли скобку, но забыли её закрыть. Компьютер не может понять, где заканчивается выражение.

**Как исправить:**
1. Проверьте все открывающие скобки — у каждой должна быть своя закрывающая.
2. Если у вас вложенные скобки, проверяйте их по порядку: закрывайте сначала внутренние, потом внешние.

**Неправильно:**
```basic
x = (5 + 3    ' не хватает закрывающей скобки
```

**Правильно:**
```basic
x = (5 + 3)   ' скобка закрыта
```

## Ошибка №9

**Что пишет компьютер (по-английски):**
`SUB structure can not contain other SUB structure`

**Что пишет компьютер (по-русски):**
Структура SUB не может содержать в себе другую структуру SUB

**Что это значит простыми словами:**
Вы пытаетесь создать процедуру (`SUB`) внутри другой процедуры. Это запрещено — каждая процедура должна быть самостоятельной.

**Как исправить:**
1. Вынесите внутреннюю процедуру наружу.
2. Каждая процедура должна начинаться с `Sub` и заканчиваться `EndSub`.
3. Процедуры не должны быть вложенными друг в друга.

**Неправильно:**
```basic
Sub Main
    TextWindow.WriteLine("Начало")
    Sub Inner   ' нельзя создавать процедуру внутри процедуры
        TextWindow.WriteLine("Внутри")
    EndSub
EndSub
```

**Правильно:**
```basic
Sub Main
    TextWindow.WriteLine("Начало")
EndSub

Sub Inner
    TextWindow.WriteLine("Внутри")
EndSub
```

**Неправильно:**
```basic

```

**Правильно:**
```basic
Goto myLabel
```

`Label with this name is already defined in this function`

**Что пишет компьютер (по-русски):**

**Что это значит простыми словами:**
Вы пытаетесь создать две метки с одинаковым именем внутри одной функции. У каждой функции метки должны быть уникальными.

```basic
Sub Test
```

## Ошибка №109

`Module must only contain procedure definitions`

**Что пишет компьютер (по-русски):**
Модуль должен содержать только определения процедур

**Что это значит простыми словами:**
В файле модуля (подключаемом через `Include`) есть код, который не является процедурой. Модули могут содержать только `Sub` и `Function`.

**Как исправить:**
1. Уберите весь код вне процедур из модуля.
2. Оставьте только объявления `Sub` и `Function`.

**Неправильно:**
```basic
' myModule.bas
x = 5   ' нельзя, должен быть только код внутри Sub
```

**Правильно:**
```basic
' myModule.bas
Sub MyProcedure
    x = 5
EndSub
```
